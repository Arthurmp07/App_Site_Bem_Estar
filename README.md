#App_Site_Bem_Estar

Este projeto é a reconstrução de um trabalho desenvolvido no ano passado.
O objetivo é criar uma aplicação voltada ao bem-estar do usuário, oferecendo planos de dieta e treino de forma personalizada.

O projeto contará com duas versões:

Versão Web: desenvolvida em PHP, Python e SQL.

Versão Mobile: desenvolvida em Java com Android Studio.
